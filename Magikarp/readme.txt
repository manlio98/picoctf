Istruzioni per completare la sfida ctf "Magikarp":

Soluzione tramite webshell
1. Avviare l'instanza e connettersi tramite ssh
2. Possibile usare solo alcuni comandi
3. Usiamo ls e notiamo tra i file una parte della flag e un file che contiene l'istruzione per ottenre la seconda parte della flag
4. Istruzione è andare nella cartella "/" e lo facciamo attraverso il comando cd /
5. Qui troviamo la seconda parte della flag e l'istruzione per la terza parte della flag
6. L'istruzione è andare nella cartella home dell'utente tramite tilde (~)
7: Qui troviamo l'ultima parte della flag.
