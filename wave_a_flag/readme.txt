Istruzioni per completare la sfida ctf "Wave a Flag":

Soluzione solamente per webshell o qualsiasi Linux computer
    - Scaricare il file in questo modo " wget https://mercury.picoctf.net/static/b28b6021d6040b086c2226ebeb913bc2/warm"
    - Prima di lanciarlo, dobbiamo renderlo eseguibile con questo comando "chmod +x warm"
    - Adesso è possibile eseguire il programma: "./warm -h > flag.txt" per ottenere il flag.
    - il flag è visibile nel file flag.txt
